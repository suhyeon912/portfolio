## 7. 배포 계획

| 항목 | 결정 사항 |
|------|-----------|
| 프레임워크 | Next.js (App Router + TypeScript + Tailwind) |
| 배포 플랫폼 | Vercel (GitHub push → 자동 배포) |
| 레포 | 개인 GitHub 공개 레포 |
| 도메인 | Vercel 기본 URL → 추후 커스텀 도메인 고려 |

### 초기 세팅 명령어
```bash
npx create-next-app@latest portfolio --typescript --tailwind --app
cd portfolio
git init
git remote add origin https://github.com/YOUR_ID/portfolio.git
git push -u origin main
# → vercel.com 에서 레포 연동 후 Deploy
```
